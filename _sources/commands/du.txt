du
==

**Name**

``duÂ - estimate file space usage``

**Synopsis**

``du [option...] [file...]``

**Description**

Summarize disk usage of each FILE, recursively for directories.

-s, --summarize 
    display only a total for each argument

-h, --human-readable 
    print sizes in human readable format (e.g., 1K 234M 2G)
