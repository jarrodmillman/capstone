.. _rm:

rm
==

**Name**

rm -- remove files or directories

**Synopsis**

**rm** [options...] {file...}

**Description**

rm removes each specified file. By default, it does not remove
directories.


-f, --force
    ignore nonexistent files, never prompt

-i, --interactive
    prompt before any removal

-r, -R , --recursive
    remove the contents of directories recursively
