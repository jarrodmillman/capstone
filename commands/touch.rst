.. _touch:

touch
=====

**Name**

touch -- change file timestamps

**Synopsis**

**touch** [options...] {file...}

**Description**

Update the access and modification times of each FILE to the
current time.


-a
    change only the access time

-m 
    change only the modification time

-t STAMP
    use [[CC]YY]MMDDhhmm[.ss] instead of current time
